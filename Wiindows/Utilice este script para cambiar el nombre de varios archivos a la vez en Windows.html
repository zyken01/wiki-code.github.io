Utilice este script para cambiar el nombre de varios archivos a la vez en Windows

por admin

Hola! volvemos otra vez con un nuevo contenido que nos gustaría que te sea de gran utilidad y solucione tus preguntas ¡Vamos a ello!

¿Necesita cambiar el nombre de varios archivos a la vez en Windows? Puede ser una tarea bastante complicada hacerlo manualmente, pero Windows admite scripts que puede ejecutar para automatizar el proceso de cambio de nombre, lo que le ahorra mucho tiempo.

Como ejemplo, considere un caso como en la imagen de ejemplo anterior, donde tiene una carpeta de cientos de imágenes, cada una con el nombre Copia de y luego una palabra o dos, como Copia de té negro.jpg.

En lugar de cambiar manualmente el nombre de cada archivo para eliminar “Copia de” o para cambiar esas palabras a otra cosa, puede ejecutar un script para hacer todo el cambio de nombre por usted.

Los programas de software y las cámaras a menudo agregan un conjunto específico de caracteres a los archivos exportados, por lo que este script es útil en esas circunstancias.

Cómo hacer el script de cambio de nombre

Un script es esencialmente un conjunto de comandos cuidadosamente elaborado para decirle a la computadora exactamente qué hacer. Aquí está el script de “buscar y reemplazar” que estamos tratando:

Establecer objFso = CreateObject (“Scripting.FileSystemObject”)

Establecer carpeta = objFSO.GetFolder (“ENTRAR RUTA AQUÍ”)

Para cada archivo de la carpeta.

sNewFile = File.Name

sNewFile = Reemplazar (sNewFile, “ORIGINAL”, “REEMPLAZO”)

si (sNewFile <> File.Name) entonces

File.Move (Archivo.Carpeta principal+ ”” + SNewFile)

terminara si

próximo

Para utilizar este script se requiere un editor de texto. El Bloc de notas, integrado en Windows, funcionará bien.

Paso 1: Abra el Bloc de notas. Puede hacer esto buscando Bloc en el menú Inicio o ejecutando el bloc
comando en el cuadro de diálogo Ejecutar (GANAR + R).

Paso 2: Copie el script exactamente como se muestra arriba y péguelo en el Bloc de notas.

Paso 3: Edite la secuencia de comandos de cambio de nombre del archivo para que se aplique a su situación particular.

Para hacer eso, necesita cambiar el texto llamado ENTRAR RUTA AQUÍ a la carpeta exacta donde se encuentran los archivos que pronto cambiarán de nombre.

Por ejemplo, tal vez desee cambiar el nombre de un grupo de archivos en una carpeta en su escritorio, en cuyo caso la ruta de su carpeta podría verse así: C: Usuarios Matt Escritorio MP3 convertidos .

Para que el script siempre se aplique a la carpeta en la que se encuentra actualmente, simplemente cambie la ruta a . . Es decir, un punto y luego una barra invertida, sin espacio. Usar el script de esta manera le permite colocarlo en cualquier carpeta y hacer que se aplique automáticamente solo a esa carpeta.

También cambia ORIGINAL
a los caracteres que desea reemplazar y eliminar REEMPLAZO para que pueda ingresar el texto que debe reemplazar los caracteres originales. En otras palabras, puede leer esta línea de la secuencia de comandos como “reemplace ESTO con ESTO.”

Nota: asegúrese de mantener las comillas en todos los casos en que las vea. Deben permanecer en la ruta de la carpeta y en la sección de reemplazo.

Paso 4: Ir Expediente > Guardar como y asigne el nombre al archivo que desee, pero asegúrese de cambiar la opción “Guardar como tipo” a Todos los archivos y anexar .vbs

al final del nombre del archivo.Paso 5

: Ahora puede cerrar el Bloc de notas y ejecutar el archivo VBS para aplicar el script. ¡Eso es! Para editar el archivo VBS para cambiar la ubicación de los archivos para cambiar el nombre, o para ajustar qué reemplazar en los archivos, simplemente haga clic derecho en el archivo VBS como ve arriba, pero en lugar de abrirlo, elija Editar

.

Cómo cambiar el nombre de archivos de forma masiva en Windows 10

Si está utilizando Windows 10, hay una función de cambio de nombre incorporada que es fácil de usar y podría ser exactamente lo que busca. Este método es único en comparación con el script anterior porque funciona incluso si los archivos tienen nombres de archivo completamente diferentes.

En otras palabras, estos dos métodos tienen casos de uso completamente diferentes. Digamos que tiene 100 archivos y cada uno tiene la palabra casa en ellos junto con otros personajes aleatorios.

Desea mantener intactos todos los caracteres, pero convierta la palabra casa en casa

. El guión es genial para eso. Sin embargo, si los 100 archivos tienen todos los nombres de caracteres aleatorios y desea que sean realmente similares, comohousepics , puede utilizar la función de cambio de nombre de Windows 10 para cambiar el nombre del primero enfotos de la casa (1) , el segundo enfotos de la casa (2) , el tercero enfotos de la casa (3)

, y así.

Aquí se explica cómo hacer esto en Windows 10:Paso 1

: Resalte los archivos a los que desea cambiarles el nombre.Paso 2 : Presione el F2 o haga clic con el botón derecho en uno de los archivos seleccionados y elijaRebautizar

.Paso 3 : Escriba el nombre de archivo que desea usar y luego presioneEntrar

.

Al instante, cada archivo seleccionado utilizará exactamente el mismo nombre de archivo. Si cada archivo usa una extensión de archivo diferente, todos tendrán el mismo nombre, pero si tienen la misma extensión de archivo, se agregará un número al final ya que dos o más archivos no pueden usar el mismo nombre de archivo en el mismo carpeta.
