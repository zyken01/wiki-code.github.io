    @kharnam https://getcomposer.org/doc/01-basic-usage.md#platform-packages

    ext-name allows you to require PHP extensions (includes core extensions). Versioning can be quite inconsistent here, so it's often a good idea to set the constraint to *. An example of an extension package name is ext-gd.

      if someone is still facing this issue, please use these below steps to fix the problem

      1. Go to the php.ini file and remove ; before
      ;extension=gd

      after

      extension=gd
      1. execute the command to change your cache
      composer require psr/simple-cache ^2.0

      Then install maatwebsiteexcel

      composer require maatwebsite/excel
