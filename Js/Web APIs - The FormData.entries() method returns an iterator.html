FormData.entries()

The FormData.entries() method returns an iterator allowing to go through all key/value pairs contained in this object. The key of each pair is a string object; the value either a string, or a Blob.

Note: This method is available in Web Workers.

Syntax

entries()

Return value

Returns an iterator.

Examples

// Create a test FormData object
var formData = new FormData();
formData.append('key1', 'value1');
formData.append('key2', 'value2');

// Display the key/value pairs
for(var pair of formData.entries()) {
   console.log(pair[0]+ ', '+ pair[1]);
}

The result is:

key1, value1
key2, value2
